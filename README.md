### Script
```$xslt
  // yarn [command]
  $> yarn start   

  - [ command ]
      "start": Dev configuration
      "build": Build folder ( /dist )
      "doc": Make Documentation
      "test": Unit test ( Karma + jasmine + istanbul )
      "lint": Typescript lint
      "e2e": E2E test ( Protractor )
```
